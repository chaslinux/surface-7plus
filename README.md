# surface-7plus
This script installs the touch drivers for the  Microsoft Surface Pro 7+. As of 2026-01-14 it has also been tested and found working with a Surface 6 running Linux Mint Cinnamon.

IMPORTANT NOTE: this script asks to enroll the driver in UEFI MOK, the password for enrolling in MOK is currently surface, but please pay attention to the end of the script
in case the MOK key changes.
